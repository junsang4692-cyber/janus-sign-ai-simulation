# Janus Sign AI Simulation

이 프로젝트는 야누스 표지판의 3×3 NIR 패턴을
AI(CNN) 기반으로 인식하는 시뮬레이션을 구현한 것입니다.

## ✔ 주요 기능
- 3×3 비트 패턴 생성 (최대 512가지)
- NIR 반사 특성 기반 이미지 합성
- CNN 모델을 이용한 패턴 분류
- 악천후 상황 모사: 부분 가림(coverage), intensity 감소
- 가려짐 비율 대비 인식 정확도 분석
- coverage–intensity 평면 분석 시각화
