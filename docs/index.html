야누스 표지판 AI 시뮬레이션

이 프로젝트는 3×3 NIR 패턴을 가지는 야누스 구조 표지판을 가정하고,
합성된 라이다 intensity 이미지를 이용해 CNN이 표지판 ID를 인식하는 과정을 시뮬레이션합니다.

1. 표지판 인식 원리

  • 표지판 내부를 3×3 셀로 분할하고, 각 셀을 고반사(1) / 저반사(0)로 코딩합니다.
  • 야누스 D/M/D 구조를 통해 가시광에서는 일반 표지판처럼 보이지만, 근적외선(NIR)에서는 3×3 패턴이 강한 대비로 나타납니다.
  • 라이다 센서는 이 NIR intensity map을 촬영하고, CNN 분류기가 해당 패턴 ID를 인식합니다.

2. AI 시뮬레이션 구성

  1. 3×3 비트 패턴 → 32×32 흑백 이미지로 변환 (합성 라이다 맵)
  2. 여러 패턴에 대해 합성 데이터를 생성하고, TensorFlow/Keras 기반 CNN으로 학습
  3. 눈/오염을 가정해 셀 내부 픽셀을 일부 0으로 만들어 부분 가려짐을 시뮬레이션
  4. 각 셀에 대해
    • 가려진 면적 비율(coverage)
    • 셀 평균 intensity (0~1)
    • threshold 기반 디지털 인식 결과(1/0)
    를 통계적으로 수집합니다.

3. 가려진 면적과 인식 성능

합성 데이터에 대한 통계 분석 결과, 고반사 셀(논리값 1)에 대해 가려진 면적이 평균 xx% 이하이고, 셀 평균 intensity가 0.yy 이상이면 약 zz% 이상의 확률로 ‘1’로 안정적으로 인식되는 것을 확인했습니다.

위의 xx, yy, zz 값은 Colab에서 시뮬레이션을 실행한 실제 결과로 교체해서 작성하면 됩니다.
